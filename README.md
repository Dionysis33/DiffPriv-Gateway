# DiffPriv-Gateway 

### Middleware Διαφορικής Ιδιωτικότητας για Μικρομεσαίες Επιχειρήσεις (ΜΜΕ)

Το **DiffPriv-Gateway** είναι μια λύση ανοικτού κώδικα που βοηθά τις επιχειρήσεις να συμμορφωθούν με τον **GDPR**, εφαρμόζοντας **Διαφορική Ιδιωτικότητα** (Differential Privacy) στα δεδομένα τους.

## Το Επιχειρηματικό Πρόβλημα
Οι ΜΜΕ συχνά δεν έχουν τους πόρους για σύνθετες τεχνολογίες προστασίας. [cite_start]Η απλή αφαίρεση ονομάτων δεν αρκεί, καθώς οι ταυτότητες μπορούν να αποκαλυφθούν μέσω συνδυασμού δεδομένων[cite: 269].

## Η Λύση μας
Αυτό το εργαλείο λειτουργεί ως "πύλη" που επεξεργάζεται στατιστικά ερωτήματα και επιστρέφει αποτελέσματα με προσθήκη **Gaussian θορύβου**, διασφαλίζοντας την ιδιωτικότητα των ατόμων[cite: 272].

## Οδηγίες Εγκατάστασης
1. Κλωνοποίηση του αποθετηρίου: `git clone https://github.com/Dionysis33/DiffPriv-Gateway.git`
2. Δημιουργία περιβάλλοντος: `python -m venv venv`
3. Ενεργοποίηση: `.\venv\Scripts\activate`
4. Εγκατάσταση βιβλιοθηκών: `pip install -r requirements.txt`
